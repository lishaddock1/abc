# 思源笔记朋友关系记录插件

一个用于管理朋友关系和互动记录的思源笔记插件。

## 功能特性

- 👥 朋友信息管理
- 📝 互动记录系统
- ⏰ 智能提醒功能
- 📊 数据分析可视化
- 🔍 快速搜索和筛选

## 安装方法

1. 下载插件文件
2. 在思源笔记中打开插件文件夹
3. 将插件文件夹复制到思源笔记的 `plugins` 目录
4. 重启思源笔记
5. 在插件设置中启用本插件

## 使用方法

### 添加朋友
1. 点击右侧边栏的"朋友管理"面板
2. 点击"添加朋友"按钮
3. 填写朋友基本信息
4. 保存即可创建朋友文档

### 记录互动
1. 在朋友管理面板点击"快速记录"
2. 选择朋友和互动类型
3. 填写互动详情
4. 保存记录

### 查看分析
1. 点击"数据分析"按钮
2. 查看联系频率、关系网络等统计信息

## 数据结构

插件使用思源笔记的文档和属性系统存储数据：

- 朋友信息存储在 `/Friends/` 目录下
- 互动记录存储在 `/Interactions/` 目录下
- 使用块属性存储结构化数据

## 开发说明

本插件使用标准的思源笔记插件开发规范，主要技术栈：

- JavaScript/TypeScript
- 思源笔记 API
- HTML/CSS
